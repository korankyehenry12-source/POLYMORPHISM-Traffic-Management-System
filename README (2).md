# POLYMORPHISM

### What is Demonstrated

- **Method Overriding** — child classes redefine `activate()` from the parent
- **Dynamic Method Dispatch** — Python selects the correct method at runtime
- **Polymorphic Loop** — one loop activates all device types without checking types
- **Extensibility** — `EmergencySiren` added without modifying existing code
